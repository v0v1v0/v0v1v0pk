countries R Documentation

Socioeconomic data for the most populous countries.

Description

Socioeconomic data for the most populous countries.

Usage

data(countries)

Format

A data frame with 42 observations on the following 7 variables.

Country

name of the country.

Popul

population.

PopDens

population density.

GDPpp

GDP per inhabitant.

LifeEx

mean life expectation

InfMor

infant mortality

Illit

illiteracy rate

Source

CIA World Factbook https://www.cia.gov/the-world-factbook/

Examples

data(countries)
summary(countries)
