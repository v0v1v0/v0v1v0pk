swissroll R Documentation

Swiss Roll Data Set Generator

Description

Computes the Swiss Roll data set of a given number of data points.

Usage

swissroll(N=1000)

Arguments

N

Number of data points.

Value

The function returns a three dimensional matrix.

Author(s)
