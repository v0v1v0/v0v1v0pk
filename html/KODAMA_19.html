mcplot R Documentation

Evaluation of the Monte Carlo accuracy results

Description

This function can be used to plot the accuracy values obtained during KODAMA procedure.

Usage

mcplot(model)

Arguments

model

output of KODAMA.

Value

No return value.

Author(s)

Stefano Cacciatore
