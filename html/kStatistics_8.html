ff R Documentation

Falling factorial

Description

The function computes the descending (falling) factorial of a positive integer n with respect to a positive integer k less or equal to n.

Usage

ff( n=1, k )

Arguments

n

integer
k

integer

Details

Run ff(n, k) to get n(n-1)(n-2).....(n-k+1).

Value

integer

the descending factorial
