knitrBootstrap R Documentation

knitrBootstrap A framework to create Bootstrap 3 HTML reports from knitr Rmarkdown. Knitr Bootstrap includes a bootstrap_document custom rendering function for use with the rmarkdown https://rmarkdown.rstudio.com/) package.

Description

You can also specify the options in your YAML front-matter, see the rmarkdown documentation for examples.

Examples

## Not run: 
library(rmarkdown)
library(knitrBootstrap)

render('input.Rmd', bootstrap_document())

#with some options
render('input.Rmd', bootstrap_document(title='Test', highlight='Github',
  theme='Slate', theme.chooser=TRUE, highlight.chooser=TRUE))

## End(Not run)
