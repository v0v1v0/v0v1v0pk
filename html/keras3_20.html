activation_relu6 R Documentation

Relu6 activation function.

Description

It's the ReLU function, but truncated to a maximum value of 6.

Usage

activation_relu6(x)

Arguments

x

Input tensor.

Value

A tensor, the result from applying the activation to the input tensor x.
