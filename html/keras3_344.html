metric_categorical_hinge R Documentation

Computes the categorical hinge metric between y_true and y_pred.

Description

Formula: 

loss <- maximum(neg - pos + 1, 0)

where neg=maximum((1-y_true)*y_pred) and pos=sum(y_true*y_pred)

Usage

metric_categorical_hinge(
  y_true,
  y_pred,
  ...,
  name = "categorical_hinge",
  dtype = NULL
)

Arguments

y_true

The ground truth values. y_true values are expected to be either ⁠{-1, +1}⁠ or ⁠{0, 1}⁠ (i.e. a one-hot-encoded tensor) with shape = ⁠[batch_size, d0, .. dN]⁠.
y_pred

The predicted values with shape = ⁠[batch_size, d0, .. dN]⁠.
...

For forward/backward compatability.
name

(Optional) string name of the metric instance.
dtype

(Optional) data type of the metric result.

Value

Categorical hinge loss values with shape = ⁠[batch_size, d0, .. dN-1]⁠.

Usage

Standalone usage: 

m <- metric_categorical_hinge()
m$update_state(rbind(c(0, 1), c(0, 0)), rbind(c(0.6, 0.4), c(0.4, 0.6)))
m$result()

## tf.Tensor(1.4000001, shape=(), dtype=float32)


m$reset_state()
m$update_state(rbind(c(0, 1), c(0, 0)), rbind(c(0.6, 0.4), c(0.4, 0.6)),
               sample_weight = c(1, 0))
m$result()

## tf.Tensor(1.2, shape=(), dtype=float32)
