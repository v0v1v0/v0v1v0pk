is.normalized-methods R Documentation

Test method

Description

Tests whether a capacity is normalized, i.e., if its value on the universal set is 1. The capacity can be given either under the form of an object of class capacity, card.capacity or Mobius.capacity.

Methods

object = "Mobius.capacity"

Returns a logical.

object = "capacity"

Returns a logical.

object = "card.capacity"

Returns a logical.

See Also

capacity-class,
Mobius.capacity-class,
card.capacity-class.

Examples

## a capacity
mu <- capacity(0:15)
is.normalized(mu)
normalize(mu)

## its Mobius transform
a <- Mobius(mu)
is.normalized(a)
normalize(a)

## a cardinal capacity
mu <- uniform.capacity(7)
is.normalized(mu)
