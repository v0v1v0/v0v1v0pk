testped1 R Documentation

testped1

Description

Sample pedigree

Usage

data(testped1)

Format

A data frame with 79 subjects in one family with the following variables.

id

subject id, unique to each ped

father

id of the subject's father

mother

id of the subject's mother

sex

1=male, 2=female

Examples

data(testped1)
testped1[1:20,]
