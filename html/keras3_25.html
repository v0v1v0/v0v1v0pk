activation_softplus R Documentation

Softplus activation function.

Description

It is defined as: softplus(x) = log(exp(x) + 1).

Usage

activation_softplus(x)

Arguments

x

Input tensor.

Value

A tensor, the result from applying the activation to the input tensor x.
