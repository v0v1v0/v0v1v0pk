config_backend R Documentation

Publicly accessible method for determining the current backend.

Description

Publicly accessible method for determining the current backend.

Usage

config_backend()

Value

String, the name of the backend Keras is currently using. One of "tensorflow", "torch", or "jax".

Examples

config_backend()

## [1] "tensorflow"
