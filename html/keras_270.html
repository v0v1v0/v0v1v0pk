k_repeat R Documentation

Repeats a 2D tensor.

Description

If x has shape (samples, dim) and n is 2, the output will have shape (samples, 2, dim).

Usage

k_repeat(x, n)

Arguments

x

Tensor or variable.
n

Integer, number of times to repeat.

Value

A tensor
