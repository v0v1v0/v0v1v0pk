activation_mish R Documentation

Mish activation function.

Description

It is defined as:

mish(x) = x * tanh(softplus(x))

where softplus is defined as:

softplus(x) = log(exp(x) + 1)

Usage

activation_mish(x)

Arguments

x

Input tensor.

Value

A tensor, the result from applying the activation to the input tensor x.
