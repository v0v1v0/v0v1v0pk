nlist-deprecated R Documentation

Smart named list

Description

This function is a wrapper around list() that assigns names to unnamed arguments based on the unevaluated expression used in the call.

Usage

nlist(...)

Arguments

...

List items, possibly named

Value

A named list.

Author(s)
